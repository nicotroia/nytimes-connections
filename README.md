## What is this

A basic React implementation of the NYTimes game "Connections"

Bootstrapped with Next.js using Typescript, Tailwind

## Getting Started

- `npm install`
- `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
